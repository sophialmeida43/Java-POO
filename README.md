# ☕ Java POO – Repositório de Estudos

Repositório destinado ao estudo e prática de **Programação Orientada a Objetos (POO)** utilizando a linguagem Java.

Os projetos foram desenvolvidos com foco na aplicação prática dos principais conceitos da orientação a objetos, como modelagem de classes, encapsulamento, manipulação de dados, criação de objetos e estruturação de código em Java.

---

## 📂 Estrutura do Repositório

### 📌 POO – Fundamentos de Programação Orientada a Objetos

Projeto base voltado para a introdução e prática dos conceitos fundamentais de POO em Java.

Neste módulo foram desenvolvidas diferentes classes com o objetivo de representar entidades do mundo real e exercitar a construção de objetos e seus comportamentos.

#### 🔹 Classes implementadas:
- **`Pessoa.java`**  
Classe que representa uma pessoa o objeto possui atributos como nome, idade e data de nascimento. A partir desses dados, o programa realiza operações simples, como cálculo da idade futura após o próximo aniversário, permitindo praticar manipulação de dados e lógica aplicada em objetos.

- **`Circulo.java`**  
Classe que representa um círculo geométrico a partir do raio informado, o programa realiza cálculos como **perímetro (circunferência)** e outras propriedades matemáticas do círculo, aplicando fórmulas básicas dentro de um contexto orientado a objetos.

- **`Produto.java`**  
Classe que representa um produto de compra o objeto contém informações como nome e preço, simulando um cenário simples de sistema de vendas, onde é possível manipular e exibir dados relacionados ao produto.


#### 🧠 Conceitos aplicados:

- Criação e instanciação de classes e objetos  
- Encapsulamento (getters e setters)  
- Atributos e métodos  
- Organização de código em pacotes  
- Entrada de dados utilizando `Scanner`  
- Modelagem de entidades do mundo real  
- Reutilização e estruturação de código orientado a objetos  

---

### 📌 ToString – Representação Textual de Objetos

Projeto voltado ao estudo da sobrescrita do método `toString()`, utilizado para personalizar a forma como objetos são exibidos no console.

O projeto utiliza as classes `Livro.java` e `Main.java`, permitindo a criação de objetos e exibição estruturada e legível das informações.

#### 🧠 Conceitos aplicados:

- Sobrescrita do método `toString()`  
- Representação textual de objetos  
- Criação e manipulação de instâncias  
- Estruturação da classe principal (`Main`)  
- Organização e legibilidade de código
