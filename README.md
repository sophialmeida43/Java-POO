# ☕ Java POO

Repositório destinado aos estudos de **Programação Orientada a Objetos (POO)** utilizando Java.

Os projetos foram desenvolvidos com foco na aplicação prática dos principais conceitos da orientação a objetos, incluindo criação de classes, objetos, encapsulamento, métodos, entrada de dados e representação textual de informações.

---

# 📂 Estrutura do Repositório

## 📌 POO

Projeto desenvolvido para prática dos fundamentos da Programação Orientada a Objetos utilizando Java.

Neste projeto foi criada a classe `Pessoa.java`, utilizada para representar objetos do tipo pessoa através de atributos e métodos.

### Conceitos aplicados:
- Criação de classes e objetos
- Atributos e métodos
- Encapsulamento
- Instanciação de objetos
- Organização em pacotes
- Manipulação de dados
- Entrada de dados utilizando `Scanner`

---

## 📌 ToString

Projeto voltado para o estudo da representação textual de objetos através da sobrescrita do método `toString()`.

O projeto utiliza as classes `Livro.java` e `Main.java`, permitindo a criação de objetos e exibição personalizada de informações no console.

### Conceitos aplicados:
- Sobrescrita do método `toString()`
- Estruturação de classes
- Criação e manipulação de objetos
- Execução principal com classe `Main`
- Organização de código em Java
- Saída de dados personalizada
